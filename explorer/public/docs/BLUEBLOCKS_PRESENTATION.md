```
│   │                         │           │                         │            │
│   └─────────────────────────┘           └─────────────────────────┘            │
│                                                                                 │
│   CRYPTOGRAPHY                          STORAGE                                 │
│   ┌─────────────────────────┐           ┌─────────────────────────┐            │
│   │                         │           │                         │            │
│   │   Encryption: AES-256   │           │   Off-chain: IPFS       │            │
│   │   Signing: Ed25519      │           │   Indexing: TimescaleDB │            │
│   │   KDF: Argon2id         │           │   State: Merkle Tree    │            │
│   │   Hashing: SHA-256      │           │   Cache: Redis          │            │
│   │                         │           │                         │            │
│   └─────────────────────────┘           └─────────────────────────┘            │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

| Component | Technology | Why |
|-----------|------------|-----|
| Language | Go | Performance, concurrency, Cosmos ecosystem |
| Contracts | Starlark | Python-like, safe, deterministic |
| Encryption | AES-256-GCM | HIPAA compliant, AEAD |
| Signing | Ed25519 | Fast, secure, small keys |
| KDF | Argon2id | Memory-hard, GPU-resistant |
| Storage | IPFS | Decentralized, content-addressed |
